# 拼车乘客微信小程序（uni-app）

Vue3 + Vite 的 uni-app 项目，面向微信小程序。

## 功能亮点

- **路线估价**：发单前调用 `/api/map/route`，展示里程、时长、基础价
- **天气动态调价**：展示天气状况、气温、倍率与附加费
- **地图预览**：估价后在首页地图显示起终点与路线
- **快捷目的地**：一键选择常用路线
- **进行中行程横幅**：首页直达追踪
- **行程列表**：全部 / 进行中 / 已完成筛选 + 下拉刷新
- **详情页**：6 步状态时间线、地图追踪司机、星级评价弹窗
- **个人中心**：行程统计、快捷入口

## 开发

```bash
cd passenger-uniapp
npm install
npm run dev:mp-weixin
```

微信开发者工具导入 `dist/dev/mp-weixin`，勾选 **不校验合法域名**。

Gateway 默认 `http://127.0.0.1:30080`（`src/config/index.ts`）。

测试账号：`passenger1` / `123456`
